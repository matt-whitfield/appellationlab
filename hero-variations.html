Hero Section Layout Variations

Exploring different approaches to hero section design for Appellation Lab's wine region storytelling platform

Variation 1

Full-Width Background with Text Overlay

Approach: Vineyard image spans full viewport width as background, with text centered and overlaid. Creates immersive, magazine-style first impression.
Best For: Maximum emotional impact, wine country atmosphere
Trade-off: Text readability requires careful gradient/overlay treatment

Variation 3

Asymmetric Layout (60/40 Text-First)

Approach: Text occupies 60% of width on left, image takes 40% on right. Prioritizes message over imagery while maintaining visual interest.
Best For: Copy-focused storytelling, emphasizing value proposition
Trade-off: Reduces visual impact of vineyard imagery

Variation 5

Stacked Layout with Minimal Overlay

Approach: Large vineyard image on top, minimal text overlay at bottom. Detailed copy lives below the image. Lets wine country imagery breathe.
Best For: Visual-first storytelling, clean modern aesthetic
Trade-off: Requires scroll to read full value proposition

Recommendation Summary

For Maximum Wine Country Atmosphere: Variations 1, 5, or 7 (full-width backgrounds with overlays)

For Balanced Readability + Impact: Variations 2 or 3 (split-screen layouts)

For Immediate Value Communication: Variation 6 (modular cards showing benefits)

Current Implementation: Variation 4 (safe choice but less impactful than alternatives)
